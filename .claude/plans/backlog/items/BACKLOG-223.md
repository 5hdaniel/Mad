# BACKLOG-223: Add Text Message Status Indicator (Like Email Status)

## Type
Enhancement / UX

## Priority
Medium

## Description
Add a status indicator for text messages (iMessage import) similar to the email connection status. Users should be able to see at a glance whether iMessages have been imported and the last import date.

## Current State
- Email status shows connection state and accuracy (TASK-1039)
- No equivalent indicator for iMessages/text messages

## Proposed Solution
Add to Dashboard or Settings:
- iMessage import status (imported/not imported)
- Last import date/time
- Number of messages imported
- Quick link to re-import

## Acceptance Criteria
- [ ] Dashboard shows iMessage import status
- [ ] Last import date visible
- [ ] Message count visible
- [ ] Status updates after import
